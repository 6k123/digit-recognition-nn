# 手写数字识别神经网络

一个基于 Python 实现的简单神经网络，用于识别 MNIST 手写数字，本项目主要用于学习神经网络基本原理。

## ✨ 特点

- 纯 Python + NumPy 实现，不依赖深度学习框架
- 包含训练集和测试集，可快速运行验证
- 代码结构清晰，注释详细，适合初学者阅读

## 📁 文件说明

提供了完整的 MNIST 数据集文件 `mnist_data.csv` 。

## 🚀 快速开始

### 1. 环境要求

- Python 3.7 或更高版本
- pip 包管理工具

### 2. 安装依赖

```bash
pip install -r requirements.txt
